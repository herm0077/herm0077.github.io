LIB2024- QA

  1. Qus: What are the three componets of Web?

    Ans: HTML, CSS and Javascript.

  2. Qus: Define version control and give one example.

    Ans: Version control software records changes to a file and allows a user to toggle between specific versions. This is useful if a user makes a mistake that they cannot identify and they have to go back to a previous version of their file.

  3. Q: What tag is used to create a paragraph in html? give an example.

    A: The p tag is used to create a paragraph in html. For example:
    <p> content </p>

  4. Q: Identify the odd one from these tags: <span>, <p>, <a>, <i>

    A: <p> is the odd one out because it is a block element, while the rest of the tags are inline.

  5. Q: What are the tags required to create a navigation list in HTML?

    A: In order you need a header tag > a nav tag > a ol or ul tag > an li tag > an a tag with a href link.

  6. Q: What is the element used to create images? Name it's attributes.

    A: The element sued to create images is the img tag. It consists of a src and can contain width and height attributes as well.

  7. Q: Identify the odd one out from these tags: <table>, <tr>, <b>, <td> and state the reason.

    A: <b> is the odd tag out because all the other tags are for creating a table in html5.
