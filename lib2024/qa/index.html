  1. Q: What are the three componets of Web?

    A: HTML, CSS and Javascript.

  2. Q: Define version control and give one example.

    A: Version control software records changes to a file and allows a user to toggle between specific versions. This is useful if a user makes a mistake that they cannot identify and they have to go back to a previous version of their file.

  3. Q: What tag is used to create a the title of a web page? Give an example.

    A: The title tag is used to create the title of a webpage. <title>Content</title>

  4. Q: What tag is used to create a paragraph in html? Give an example.

    A: The p tag is used to create a paragraph in html. For example: <p> content </p>

  5. Q: Identify the odd one from these tags: <span>, <p>, <a>, <i>

    A: <p> is the odd one out because it is a block element, while the rest of the tags are inline.

  6. Q: What are the tags required to create a navigation list in HTML?

    A: In order to create a navigation list you need a <nav> tag, a <ol> or <ul> tag, an <li> tag and a <a> link, in this order.

  7. Q: What is the element used to create images? Name it's attributes.

    A: The element used to create images is the <img> tag. It consists of a src and an alt, it can also contain width and height attributes as well.

  8. Q: Identify the odd one out from these tags: <table>, <tr>, <b>, <td> and state the reason.

    A: <b> is the odd tag out because all the other tags are used to create a table in html5, while the <b> tag is used to bold text.

  10. Q: What is the use and of "name" and "value" attributes? Give 2 examples.

    A: In an HTML form, the name attribute is used to connect an <input> to a <label>, while the value attribute is used to give an element a fixed value. For example:

    Name: <label> for="colour"</label>
    <input type="color" name="colour" id="colour">
    The name attribute matches the for attribute to connect the input element to the label element.

    Value: <input type="email" name="email" id="email" value="www.example@value.com">
    In this case the value attribute is used to give the email input a fixed value of "www.example@vlue.com". A user filling out this form would not be able to change this value and it would be sent to the server as is.

  13. Q: Create a CSS block to center an element both horizonally and vertically that has position:absolute and has a width:200px and height:200px.

    A: p {

    position:absolute;
    width:200px;
    height:200px;
    margin-top:-50%;
    margin-left:-50%;

    }

  14. Q: Create a CSS keyframes block called "spin" that rotates an element.

    A: @keyframes spin {

    from {
    transform:rotate(0deg);
    }
    to {
    transform:rotate(360deg);
    }

    }

  15. Q: Create a CSS animation block to rotate an element that has the following properties:

    • Class "rotate-me"
    • It rotates 5 times at intervals of 0.5s with "linear" transition using animation keyframes "spin".

    A: .rotate-me {

    animation:
    spin 0.5s linear 5;

    }
